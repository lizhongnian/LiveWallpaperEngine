# LiveWallpaperEngine

## 新版本正在开发，可用版本请查看tag最新版
---

## 定位:
  动态壁纸最核心的那层功能

## 愿景：
提供公用api，上传到nuget，让所有程序都可以成为壁纸。。。

## 功能目标：
- [x] 嵌入桌面
- [ ] 鼠标事件转发
- [x] 支持多屏幕
- [x] 内置播放器
- [ ] 内置浏览器
